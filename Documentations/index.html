Quick-Start

Medians School Routes - PHP & FLUTTER

Features

Check out some of features that we have in this item.

Routes management

Control the available Routes for the Cars and related Driver .

Pickup & Destination Locations

Each parent can add pickup & destination for his student.

Vehicles Management

Control the Vehicles and connect with its Driver & Route.

Trips Monitoring

Parent & Administrator can track the vehicle in realtime.

Pickup days

Set the pickup or school days that the student should be picked-up.

Real-time Notifications

Using One-Signal provider to send realtime notifications to Parent & Driver.

Multi-language

Manage the control panel and App content in Multi-language

Events

Events & Articles system with images to show at the APP.

Custom API

API to manage the mobile APPs integration.

Lite Web installer

Lite Installer to install the script and set the database configuration

Google Map & locations

APIs for Google Map should be set at the Dashboard & Flutter APP.

Ticketing & Support

Ticketing system to send complaints/feedbeck from users with comments.

RTL support

Support RTL direction with pre-built CSS for RTL languages
