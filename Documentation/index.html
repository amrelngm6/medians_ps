Introduction

Medians School Routes bookings system

Thanks for choosing us, Medians Team.

Medians School Routes - Is a complete solution that provide transportation service with realtime Monitoring with GPD tracking which provide two versions for Driver & Parents. Also it has Admin Panel that can mointor and control all Routes, Pickup-Locations, Destinations, Patients, Students and Vehicles with its Driver.

You can ask for support through our profile at CodeCanyon , @AmrMeRo.

Features

Check out some of features that we have in this item.

Routes management

Control the available Routes for the Cars and related Driver .

Pickup & Destination Locations

Components

Components of the dashboard that built in Vue

Each controller has its own components which is loaded from the attributes that is being set at /app/admin/vue.html.twig which is has the route an the component name that should be loaded.

Class Description
navbar

Has the Logo, Notification & Language switch and also the user info

SideMenu Display the side menu links and hideable from the navbae icon
Dashboard Default page and has its components like Charts and cards Blocks also has static table data
system_settings Display settings for Master or system owner
users Display users or branches owners list
customers Display customers that made bookings at branches
notifications Display notifications log
notifications_events Manage when notifications should be sent for each model actions

/*
|--------------------------------------------------------------------------
| Component example
|--------------------------------------------------------------------------
|
| Import datatable actions to append as extra column in data-table rows.
| Only used for actions row like Edit/Delete/...
|
*/

import dataTableActions from './includes/data-table-actions.vue';

export default 
{
    components:{
        dataTableActions,
    },
    name:'Branches',
    data() {
        return {

            // Absolute url + component route + ( load As Json ) parameter
            url: this.conf.url+this.path+'?load=json',

            /**
            * This Object properties
            * Should be matched with the 
            * API response 
            */
            content: {
                title: '',
                users: [],
                items: [],
                columns: [],
            },

            activeItem:{},
            showAddSide:false,
            showEditSide:false,
            showLoader: true,
        }
    },



    computed: {

        /**
        * Optional for components with Data-Table only 
        * 
        */
        bindings() {


            /**
            * Optional for appending actions to the 
            * Datatable row
            */
            this.content.columns.push({
                    key: this.__("actions"),
                    component: dataTableActions,
                    sortable: false,
                });

            return {

                columns: this.content.columns,
                data: this.content.items
            }
        }
    },


    /**
    * Default props for 
    * all dynamic Components 
    */
    props: [
        'path',
        'lang',
        'setting',
        'conf',
        'auth',
    ],
    mounted() 
    {

        /**
        * Load the data after loading
        */
        this.load()
    },

    methods: 
    {
        /**
         * Check if the user is Master
         */
        isMaster()
        {
            return (this.auth && this.auth.role_id == 1);
        } , 


        /**
        * Used for data actions  
        */
        handleAction(actionName, data) {
            switch(actionName) 
            {
                case 'view':
                    // window.open(this.conf.url+data.content.prefix)
                    break;  

                case 'edit':
                    this.showEditSide = true; 
                    this.showAddSide = false; 
                    this.activeItem = data
                    break;  

                case 'delete':
                    this.$parent.delete(data, 'Branch.delete');
                    break;  
            }
        },

        /**
        * Load the data through the API
        * Send the response to update the
        * components values
        * 
        */
        load()
        {
            this.showLoader = true;
            this.$parent.handleGetRequest( this.url ).then(response=> {
                this.setValues(response)
                this.showLoader = false;
            });
        },
        

        /**
        * Set the response to ( content ) 
        */
        setValues(data) {
            this.content = JSON.parse(JSON.stringify(data)); return this
        },

        /**
        * Used to translate the content
        * emits the App.vue function
        *
        */
        __(i)
        {
            return this.$root.$children[0].__(i);
        }
    }
};
</script>
